# Visão do Projeto

## 1. Descrição do Sistema

O **GranaFácil** é um sistema de gestão financeira pessoal que permite ao usuário registrar e organizar suas movimentações financeiras mensais.

A aplicação foi desenvolvida como uma **API REST utilizando ASP.NET Core**, com foco em boas práticas de arquitetura, separação de responsabilidades e modelagem de domínio.

O sistema permite estruturar informações financeiras como contas, entradas, gastos, metas e reservas, oferecendo uma base organizada para controle financeiro.

---

## 2. Objetivo do Projeto

O objetivo do projeto é desenvolver uma aplicação backend estruturada que demonstre conhecimentos em:

- Desenvolvimento de APIs REST
- Arquitetura em camadas
- Modelagem de domínio
- Persistência de dados com Entity Framework Core
- Organização de código voltada para aplicações reais

Além disso, o projeto serve como base para estudo e evolução de funcionalidades relacionadas à gestão financeira.

---

## 3. Problema que o Sistema Resolve

Muitas pessoas utilizam planilhas ou anotações dispersas para controlar suas finanças, o que dificulta a organização e visualização clara das movimentações financeiras.

O GranaFácil busca centralizar esse controle em um sistema estruturado, permitindo que o usuário registre e acompanhe suas movimentações de forma organizada.

---

## 4. Público-Alvo

O sistema é voltado para usuários que desejam:

- controlar suas finanças pessoais
- organizar receitas e despesas
- acompanhar sua situação financeira mensal
- planejar metas financeiras

---

## 5. Escopo Geral da Solução

O sistema permite que cada usuário possua seu próprio ambiente financeiro, onde pode registrar e consultar informações relacionadas à sua vida financeira.

Entre os principais recursos do sistema estão:

- registro de contas
- registro de entradas financeiras
- registro de gastos
- organização financeira por mês
- criação de metas financeiras
- criação de reservas
